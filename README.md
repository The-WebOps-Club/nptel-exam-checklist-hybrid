Hybrid app for NPTEL exam checklist

## Gettin started
* Install `nodejs` and `npm`: https://nodejs.org/en/download/package-manager/
* Install `ionicframework`: http://ionicframework.com/docs/guide/installation.html
* Clone the repo
* cd to the directory
* Run `ionic serve`
